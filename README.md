## 环境依赖

我的环境： **python2.7**，除此之外，还需要额外安装两个库：

```bash
pip install requests
pip install prettytable # 美美地打印表格
```



## 适用人员

科大软件学院研究生



## 使用方法

- 将项目克隆到本地

  ```bash
  git clone https://github.com/jianhuchen/Course-Choosing-SSE-USTC.git
  ```

- 修改配置文件`conf.py`

- 在项目根目录运行：

```shell
python main.py
```
